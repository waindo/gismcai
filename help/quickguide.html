Quick Guide for MCA-Indonesia
Dashboard Mapping Application

 

Summary

This quick guide will walk you through MCA-I dashboard mapping application. It is meant for non GIS specialist looking to get started using the application.

Dashboard Mapping Components

There are two main components in the application, namely Navigation Window and Map Window (Fig. 1).  Navigation window contains Main Navigation, Layer Description, Basemap and Tools. Main Navigation includes Layer Category and Location tabs. Layer Category contains a list of layers that are available in the application. Location tabs is used to navigate map window according to province, district, and landscape. Map window is used to display layers, which are active/checked in Layer Category tab.


Fig. 1. Dashboard mapping has 2 main components, i.e. navigation window and map window.

Layer Description is located below the Main Navigation. It provides layer information such as short description of a layer and data source. Basemap is used to change the background of Map Window. Current default is using Ocean. Tools contain several facilities/tools such as tool to measure distance, export Map Window (inprogress), and Searching an attribute.

Exercise 1. Understanding dataset

Currently, the dashboard mapping application has two datasets, i.e. Jambi and West Sulawesi Province. Jambi dataset mostly obtained from Satgas REDD+ which has cooperation with UNDP -  REDD+ Programme. The data was collected under a project called Provision of Baseline Data and Cadastral Maps. West Sulawesi dataset is obtained from previous MCC projects, i.e. District Readiness Assessment (Abt Consultant) and Feasibility Study (NREL). This exercise will explain how to set the data to a specific province before displaying the data.

  •  Click on the Home button to view Indonesia extent and refresh Map Window

  • Click on Layer Category tab to find layer table of contents
  • On the Province dialog box set to Jambi to set the data to Jambi Province

  • Click Select
  • Click on the check box in front of layer Capital District (under Administrative) to display the capital of the district

Map Window will only display the capital of the district in Jambi Province.

  • Uncheck box in front of layer Capital District
  • Click on the Home button to view Indonesia extent and refresh Map Window
  • On the Province dialog box set to West Sulawesi to set the data to West Sulawesi Province
  • Click Select

Now, Map Window will only display the capital of the district in West Sulawesi Province. Setting up the dataset is needed before user display specific province data set. This setting up is provided because the data set between the two provinces have been different. There is a problem on data availability in Indonesia, where specific spatial data may available in a province but may not available in other provinces. To understand the different layer available between Jambi and West Province do the following steps:

  • Click home button
  • On the Province dialog box set to Jambi to set the data to Jambi Province
  • Click Select
  • Click on the check box in front of layer Carbon Stock 2011 (MoF)
  • Uncheck Layer Carbon Stock 2011 (MoF)
  • Change dataset at Layer Category to West Sulawesi
  • Click Select
  • Can you find Carbon Project category??

The Carbon Project category only available in Jambi dataset, therefore, user cannot find the same information for West Sulawesi province.

Exercise 2.  Displaying working area of MCA-Indonesia

In this exercise, users will be guided to display the working are of MCA-Indonesia project. There are three projects managed by MCA-Indonesia, i.e. Green Prosperity Project (GP), Procurement Modernization Project (PMP), and Community-based Health and Nutrition to Reduce Stunting Project (CHNRSP).

  • Click on Layer Category tab to find layer table of contents
  • Click on the check box of Green Prosperity Project to activate/display the starter districts of GP project
  • Do the same to display PMP and CHNRSP on Map Window
  • Click on the Legend menu to display the corresponding legends of the active layers

Map Window shows the starter district of GP project, which indicated by green color, while the working area of PMP and CHNRSP in district level represented by different types of hatching (see Fig. 2). The Legend will display only the active layer. If you display many layers, you have to scroll down the legend menu to find the corresponding legend.


Fig. 2. Legend menu is used to display legend of the active layer.

We are going to continue our exercise by zoom in to Jambi province

  • If Legend menu still open, click on the Legend menu to close it
  • Click on Location tab
  • Click on the radio button in front of Province, then select Jambi in the drop box of province

  • Click Zoom To direct Map Window to Jambi province. Your Map Window should display the same as Fig. 3

Map Window displays MCA-I project with geographic extend directing to Jambi Province. It also shows the projects located in the neighboring district of Jambi Province. District with two (or more) symbols indicate that the district has two (or more) projects, respectively. For example, Muaro Jambi has two MCA-I projects, i.e. GP and PMP, which are indicated by green color (GP) and vertical hatch (PMP). Banyuasin (south Sumatera) has also two projects, i.e. PMP and CHNRSP, which are indicated by vertical and horizontal hatch, respectively.

Fig. 3.  Location tab is used to zoom in into specific area (province, district, and landscape). The figure shows that MCA-I has two projects in Muaro Jambi (i.e. GP and PMP) and Banyuasin (i.e. PMP and CHNRSP). 

When you are not sure where the boundary of Jambi Province is, you can display it using the following step:

  • Click on the Layer Category tab to see layer table of content
  • On the Province dialog box set to Jambi to set the data to Jambi Province
  • Click Select
  • Click on the check box of layer District Boundary (under Administrative category) to display the district boundary.

  • Click on the check box of layer Capital District (under Administrative category) to display the capital of each district in Jambi province. Your Map Window should display the same as Fig. 4


Fig. 4. Administrative category provides layers related to administrative issue such as administrative (district, sub district, village) boundary and city/town/village location

GIS has special function. Not only it helps you to know where the location is, it also allows you to know what information in a specific location. This information is called an attribute.

  • Click on a district polygon, the attribute of that particular polygon will pop-up as in Fig. 5
  • Click on the different district polygon to display their attributes (To view the attributes of Capital District, first, you have to un-display/uncheck District Boundary, then click on red points of the Capital District)


Fig. 5. Attribute data provide information that is available in a spatial data.

Very often people asking where the (spatial) data come from. We provide Layer Description, which is located below the Main Navigation to provide you information (short description and data source) on the layer that is active/being displayed.

  • Click on the Layers Description to find short description and data source of the active layer

If you are still displaying District Boundary, Layer Description tells you that this data obtained from Biro Pusat Statistik (BPS)

  • When you want to see the layer table of contents back, click on the Main Navigation
  • Click on the Home button to return to Indonesia extent

  • Uncheck all the active layers to finish this exercise

Exercise 3.  Assessing forest encroachment in Merangin

In this exercise, user will be guided to see a sign of forest encroachment using this application. We will use forest status and landcover layers. Both spatial data were obtained from Ministry of Forestry.

  • Click on the Location tab
  • Click on the radio button in front of Province, then select Jambi in the drop box of province
  • Click Zoom To
  • Click on the Layer Category tab to see layer table of content
  • Click on the check box of layer District Boundary to display the district boundary
  • Click on the check box of layer Forest Status (under Forestry category) to display the district boundary

  • Click on the Legend menu to display the Legend of Forest Status Layer
  • You may also click on a particular polygon to know its forest status

Forest status is classified into 5 classes. Originally, for Jambi province it has 9 classes. Cagar Alam (CA), Taman Nasional (TN), Taman Hutan Rakyat (Tahura), and Taman Wisata Alam (TWA) are grouped as Conservation Area to simplify the symbol. As you may have guess, this layer is obtained from Ministry of Forestry (MoF).

  •  Click on Layer Description to check who did produce this data

You may find that in some area, the district boundary cannot be seen because it covered by forest status polygon. To solve this problem, we provide transparency level that can be used to display layers, which are overlaid. 

  • Click and hold on the transparency bar below Forestry category, move it from right to left to see district boundaries that are blocked by forest status polygon

 We are going to focus on Merangin district to assess the forest status with the existing landcover

  • Click on Location tab
  •  Click on the radio button in front of District, then select Merangin in the drop box

  • Click Zoom To
  • Click Layer Category tab to get back to the list of layers
  • Under Landcover category, click on Merangin to display the landcover of Merangin
  • Click on Layer Description if you want to know who did produce this data

After displaying the landcover, you cannot see forest status layer in Merangin because it is blocked by landcover layer.

  • Click and hold on the transparency bar below Landcover category, move it from right to left to see forest status boundaries that are blocked by landcover polygon

Using the transparency bar you are able to check that Conservation Area is not forested area in fact it has been used as agriculture. In other words you may see sign of encroachment to conservation area in a specific area (see Fig. 6.)


Fig. 6. Sign of encroachment can be check visually by overlaying forest status and landcover. 

You can zoom in to that particular area in order to have a better view using Zoom In tool.

  • Click on Zoom In menu above the Legend. Then click and drag to create rectangle area on the area of interest.

  • You may have to try a couple of times if you are not familiar with GIS software before. If your zoom in result is not displaying what you want, you can return to the previous display by clicking on Prev Entent menu.

Our Zoom In result is shown in Fig. 6b.


Fig. 6b. Zoom In to the area with the sign of encroachment

Exercise 4. Measuring distance between objects

This exercise will walk through the use of a tool to measure the distance between renewable energy sources, i.e. Microhydro, to the closest transmission line (on grid). This distance can be important if the owner of Microhydro station would like to sell electricity, which they produce, to the state (PLN). This exercise assumes that you have already practiced using exercise 1 and 2.

  • Uncheck all the layer
  • Using Location tab, navigate Map Window to Merangin district if you have not done so
  • Display layer District Boundary, this is just to make sure that you are viewing the right district
  • Display layer RE Microhydro (Merangin) under Energy category
  • Display layer Main Road under Infrastructure category
  • Display layer Transmission Line under Energy category

Your Map Window should be the same with Fig. 7a. We are going to focus on the specific area where the Microhydro station and its closest transmission lines are located. This area of interest is shown as dashed rectangle in Fig. 7a.


Fig. 7a. Map Window showing road, Microhydro, and transmission line in Merangin. The dashed rectangle is the area of interest in this exercise.

  • Using Zoom In menu, zoom to the area of interest as shown in Fig. 7a

Your Map indo should be the same with Fig. 7b. We would like to measure distance between the closest transmission lines to the Microhydro station. This distance can be important if the owner of Microhydro station would like to sell electricity, which they produce, to the state (PLN).    

  • Undisplay District Boundary
  • Click on Tools menu to find Distance tool


Fig. 7b. This exercise will measure distance between the closest transmission line and Microhydro station using Distance tool

  • Click on the Distance tool button
  • Set distance unit to Kilometers
  • Click from the closest transmission line to start measuring the distance. The green flag indicate that you are ready to measure the distance
  • Click and follow the main road to the Microhydro station. You need to make several clicks when following the main road
  •  Double click when you reach the Microhydro station

Your display should be the same as Fig 8. Several green flags show the path that you measured. The measured distance is about 37 km. This distance is not feasible for PLN to extend their transmission lines to the Microhydro station.


Fig 8. Distance tool measures distance between objects. It shows the measured path and distance.

·         When you have finished, click on Distance button to remove the measured path

·         Uncheck all layers to finish this exercise

Exercise 5. Finding village location

One of the tools that we provide in this dashboard mapping is Searching tool. This tool is useful when user would like to locate Map Window to a village location but he/she only knows about the name of the village. For example, user only knows that one of the GP project is located in Rantau Suli but he/she does not know where the village is.   Location tab cannot help as it only locates to province, district, and landscape level. This exercise will show how to use Searching tool to locate Map Window to a village. This exercise assumes that user has completed exercise 1-3, therefore he/she is already able to display specific layer as instructed below.

  • Locate your location to Jambi Province

  • Please display District and Village Boundary

Your display should be the same with Fig. 9.


Fig. 9. Map Window displays District and Village boundary.

  • Click on Tools

  • In the box of Layer Data, type “rantau suli” (without upper stroke) then click search

 

The record which matches with the search criteria will be shown in the box below the search button. While the polygon which matches with the search criteria will be highlighted with cyan color in Map Window.

 

  • Click on the selected record below the search button to zoon in to the selected village

  • If the zoom in is too close, zoom out using your mouse roller. Perhaps you need different direction of your mouse roller to find the suitable view.
